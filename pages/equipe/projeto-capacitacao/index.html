    Projeto de Capacitação

    Atividade de Capacitação: React e Git

    Objetivo: A atividade foi projetada para desenvolver as habilidades da equipe em React e Git, promovendo práticas de programação colaborativa e incentivando a criatividade no desenvolvimento de aplicações web. Como parte desse treinamento, cada membro foi desafiado a criar uma página individual que refletisse seu estilo e preferências pessoais.

    Descrição da Atividade: Os participantes foram convidados a construir uma página de apresentação pessoal, utilizando React. A proposta inicial incluiu a utilização do Figma para planejar o design da página antes de implementá-la em código, proporcionando uma experiência mais rica e bem estruturada para o projeto. Além disso, as páginas desenvolvidas foram avaliadas em um concurso interno, com prêmios para as três melhores construções.

    Execução: Os integrantes seguiram os passos abaixo para realizar a atividade:

    1. Configuração Inicial:
    2. Instalação do Git e do Node.js para gerenciar o repositório e as dependências do projeto.
    3. Clonagem do repositório no GitHub e navegação até a pasta sentinelas-membros.

    4. Instalação das dependências com o comando npm install e execução do projeto com npm start.

    5. Desenvolvimento:

    6. Criação de uma branch dedicada para cada participante, onde o desenvolvimento foi realizado.

    7. Implementação das páginas seguindo boas práticas de programação e revisão de Pull Requests (PRs) entre os membros.

    8. Critérios de Avaliação:

    9. Funcionalidade: Páginas sem erros e que atingissem os objetivos propostos.
    10. Arquitetura do React: Uso de uma estrutura de componentes bem definida.
    11. Qualidade do Código: Organização e legibilidade do código.
    12. Design e Experiência do Usuário: Apelo visual e responsividade.
    13. Originalidade: Aplicação de elementos criativos e interativos.
    14. Correção de PRs: Participação na revisão de códigos de outros membros.

    15. Documentação: Inclusão de informações claras no README.md.

    16. Pontos Extras:

    17. Uso de hooks como useState e useEffect, bem como de outros recursos avançados do React.
